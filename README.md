# Bank Marketing Prediction

## Objective
To predict whether a customer will subscribe to a term deposit using machine learning classification models.

## Dataset
Bank Marketing Dataset from UCI Machine Learning Repository.

## Models Used
- Logistic Regression
- Random Forest Classifier

## Evaluation
- Accuracy
- Confusion Matrix
- Classification Report
- ROC Curve
- LIME Explanation

## Result
Random Forest performed better than Logistic Regression.

## Conclusion
The Random Forest model was selected as the best-performing model for this classification task.
